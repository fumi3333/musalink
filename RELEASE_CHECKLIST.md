# 本番リリース前チェックリスト (Production Release Checklist)

本番環境に移行する前に、必ず以下の項目を元の設定に戻してください。

## 1. セキュリティ設定の復元
- [ ] **自画自演購入の制限を戻す**
  - ファイル: `app/transactions/new/page.tsx`
  - 内容: `{false && userData?.id === item.seller_id ...}` の `false &&` を削除して制限を有効化する。
- [ ] **デバッグ用ログインボタンの非表示**
  - 本番ビルド(`NODE_ENV === 'production'`)では自動的に消えますが、コード自体を整理することを推奨。

## 2. Stripe設定
- [ ] **APIキーを本番用(live)に切り替え**
  - `.env` および Firebase Config を `sk_live_...` に更新。
- [ ] **テスト用カード番号の無効化**
  - ドキュメントや説明からテスト用番号（4242...）を削除。

## 3. メール・通知
- [ ] **テスト送信用の宛先固定を解除**
  - 特定のテストアドレスのみに送信するロジックがないか最終確認。
